Technical Integration Guide

Verbolia proposes a cloud-based solution to generate highly optimized SEO landing pages directly on the domain of its customers.

The solution takes the customer data feed in input and will identify the keywords for which new landing pages should be created in order to attract more SEO and SEA traffic. For each of these keywords, a page is created, optimized in terms of performance and content, and automatically linked in the architecture of the client’s site.

In order for the solution to be implemented on a client’s website, 3 technical steps need to be taken:

Data Feed Integration

Ensure up-to-date feed(s) to generate the most relevant landing pages.

Reverse Proxy Redirection

Set up your reverse proxy

Internal Linking Configuration

Install the linking script on your pages to link Verbolia pages
