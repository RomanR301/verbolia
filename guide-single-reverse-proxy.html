Reverse Proxy Redirection

In order to display the pages directly on the domain name of the client, a reverse proxy redirection must be set up. A specific url pattern has to be agreed upon, in the example below: https://client.com/v/*. All requests sent to urls matching the pattern have to be reverse proxied to the Verbolia servers, in this case: client.backend.verbolia.com/v/*, see below.
